# MyTodora

MyTodora is a simple and responsive to-do list web application that helps users organize and manage their daily tasks efficiently.

## Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 💾 Automatically saves tasks in browser local storage
- 📱 Responsive design for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3  
- JavaScript (Vanilla JS)

## Project Structure

```
mytodora/
│── index.html      # Main HTML file
│── style.css       # Styling for the application
│── script.js       # JavaScript functionality
│── README.md       # Project documentation
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/mytodora.git
```

2. Navigate to the project folder:

```bash
cd mytodora
```

3. Open `index.html` in your preferred web browser.

No additional installation or dependencies are required.
